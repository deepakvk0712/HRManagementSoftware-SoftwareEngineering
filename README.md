# HRManagementSoftware-SoftwareEngineering
Creating a full stack application which will have the capability to manage employees, payroll etc.

# Tech Stack
* Front end - Vue js
* Back end - GoLang
* DB - SQLite
* Using HTTP Rest API for communication.
